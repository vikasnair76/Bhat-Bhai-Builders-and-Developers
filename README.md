# Bhat Bhai Builders & Developers (Website)
A responsive business website built with **React + Vite** for showcasing services, projects, and contact information.


## Features
- Modern, responsive UI
- Pages/sections: Home, About, Services, Projects, Contact
- Reusable React components
- Fast build & hot reload using Vite

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Project Structure
src/
  components/
  pages/ (if you have)
  assets/
  App.jsx
  main.jsx
vite.config.js

### Clone the repo
git clone https://github.com/vikasnair76/Bhat-Bhai-Builders-and-Developers.git
cd Bhat-Bhai-Builders-and-Developers

### Install Dependencies
   npm install

###  Run the project
    npm run dev

### Build for Production
npm run build

npm run preview
